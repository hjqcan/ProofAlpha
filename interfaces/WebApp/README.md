# Autotrade Web App

Browser-based React + Vite control room for `interfaces/Autotrade.Api`.

```powershell
npm install
npm run dev
```

The app defaults to `http://localhost:5080` for the API. Override with `VITE_API_BASE`:

```powershell
$env:VITE_API_BASE="http://localhost:5080"; npm run dev
```
