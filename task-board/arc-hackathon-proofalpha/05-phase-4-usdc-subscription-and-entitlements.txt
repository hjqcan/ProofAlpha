===========================================

Intent
------
Make USDC payment on Arc unlock a real ProofAlpha capability. This is the
commercial core of the hackathon project.


Accepted Scope
--------------
- StrategyAccess contract integration.
- Subscription plan model.
- Entitlement mirror in backend.
- Manual refresh/sync command.
- API read model.
- Tests for entitlement gating.


Out of Scope
------------
- Production billing.
- Custody.
- Mainnet token handling.
- Real tax/accounting.


Subscription Plans
------------------
Plan fields:

- planId
- strategyKey
- planName
- tier
- priceUsdc
- durationSeconds
- permissions
- maxMarkets
- autoTradingAllowed
- liveTradingAllowed
- createdAtUtc

Permissions:

- `ViewSignals`
- `ViewReasoning`
- `ExportSignal`
- `RequestPaperAutoTrade`
- `RequestLiveAutoTrade`
- `PublishSignal`
- `RecordSettlement`

Acceptance Criteria
-------------------
1. A wallet can have active or expired access mirrored in backend.
2. API can answer whether the wallet may view a strategy signal.
3. Subscription tx hash can be linked in WebApp or artifact.
4. Testnet/mock USDC distinction is explicit.
5. Entitlement logic is not only visual; at least one backend path uses it.
