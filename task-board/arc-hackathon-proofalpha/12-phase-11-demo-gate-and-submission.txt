Demo Runner
-----------
Create:

```powershell
.\scripts\run-arc-hackathon-demo.ps1
```

The script should:

1. Build contracts.
2. Run contract tests.
3. Build .NET.
4. Run targeted ArcSettlement and builder attribution tests.
5. Build WebApp.
6. Seed or locate one demo opportunity.
7. Publish signal proof.
8. Sync subscription entitlement.
9. Show access denied and access allowed JSON.
10. Execute or simulate Paper order with builder evidence.
11. Record performance outcome.
12. Record revenue settlement.
13. Export `demo-summary.md`.
